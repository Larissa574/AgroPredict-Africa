# AgroPredict-Africa


Machine learning project for predicting agricultural yield in West Africa using multi-source data:

- FAOSTAT (agriculture)
- OpenMeteo (climate)
- ISRIC SoilGrids (soil data)

## Goal
Predict crop yield using environmental and agricultural variables.

## Models
- Linear Regression
- Decision Tree
- Random Forest

## Workflow
Data extraction → Integration → Preprocessing → Modeling
